# pyfonizer

File sorting by type written in Python

## Requirements

`python >= 3.4`

## Usage

*nix / MacOs

``` text
python3 pyfonizer.py [path to directory]
```

Windows

``` text
python pyfonizer.py [path to directory]
```

## Install

``` text
git clone https://github.com/C0DIC/pyfonizer
cd pyfonizer
```

[Usage](#usage)

## Examples

``` console
$ chmod +x pyfonizer.py
$ ./pyfonizer.py ~/Downloads
```
